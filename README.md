# Online Retail Data Cleaning with Python
This project demonstrates basic data cleaning and preparation using Python and Pandas. The goal is to transform a raw transactional dataset into a structured dataset suitable for analysis or dashboarding.

# Tools Used

- Python(Pandas)

- Jupyter Notebook

# Dataset

- Dataset sourced from the [UCI Online Retail Dataset](https://archive.ics.uci.edu/dataset/352/online+retail)

- Contains transactional sales data from a UK-based online retailer between 2010 and 2011

- 541,909 transaction records across 8 columns

# Data Preparation

- Identify missing values and duplicates

- Removing duplicate records

- Removing invalid transactions

- Extracting Year and Month from the transaction timestamp

- Creating a Revenue column

- Generate the cleaned dataset used in the project

# Output

- A sample of the cleaned dataset is available in: line-retail-data-cleaning-with-python/cleaned-data

# Purpose

This repository demonstrates the use of Python and Pandas for data cleaning and preparation, a common step in the data analytics workflow before analysis in SQL or business intelligence tools.
